To evaluate the recharge term is possible to download the zipped folder demo_and_executable.
The code contains the input for a demonstration case in the Po basin.
It is possible to extract all the contents of the folder and, by adding the missing inputs and following the instructions of the Main jupyter notebook, it is possible to execute the program. The only missing files are the data for the evapotranspiration model. They must be downloaded and converted using the code in the meteo_data repository.
The output of the meteo_data code is a zipped folder that must at must be placed in the precip folder before running Main.ipynb.
